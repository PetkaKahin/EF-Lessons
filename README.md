# 1 Модуль

## Установка
```bash
git clone https://github.com/PetkaKahin/EF-Lessons.git
cd EF-Lessons
make up
```

## Описание

Перейти в `http://localhost/` там будут результаты задания (из-за задания на производительность страница подгружается долго) <br>
Для мест, где обязательно использовать CLI написал команды для запуска скриптов
